# H5 移动端风格指南

## 1. 总体口径

MVP 前端是 H5 移动端网页，体验接近微信小程序。前端技术栈为 Vue 3 + Vite + TypeScript + Vue Router + Pinia + Vant + Axios。

## 2. 页面基调

| 元素 | 风格要求 |
|---|---|
| 背景 | 浅灰页面背景 |
| 内容区 | 白底为主 |
| 信息容器 | 圆角卡片承载团购、订单、地址、会员卡等信息 |
| 商品图 | 商品图突出，优先展示封面 |
| 价格 | 价格比描述和标签更突出 |
| 信任信息 | 团长头像、团长名称、店铺名称、基础信任标签明显可见 |

## 3. Vant 使用口径

优先使用 Vant 的移动端组件承载表单、弹窗、列表、Tab、底部栏、按钮、空状态和反馈提示。表单页包括创建店铺、发布团购、地址填写和发货填写。

## 4. 底部 Tab

底部 Tab 承载首页、订单、个人中心等主入口。需要登录的入口应触发全局登录拦截，登录成功后恢复原操作。

## 5. 卡片流

首页团购列表、团长主页团购列表、订单列表、会员卡列表使用移动端卡片流。商品卡片优先展示商品图、标题、价格、销量、团长头像和店铺信息。

## 6. 详情页底部操作栏

团购详情页底部固定操作栏承载订阅、跟团购买等核心操作。跟团购买需要登录，未登录时触发登录页或登录弹窗。

## 7. MVP 不建设内容

MVP 不建设平台后台，不为桌面端单独设计复杂布局，不实现非 MVP 能力入口。
